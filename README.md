# rhino-consumer
Project for testing rhino, have setup already for react-query, react-hook-forms, react-router, axios.

Install dependencies:
yarn

Start the project:
yarn run dev

Initialize reno:

npx rhino -h

npx rhino -i

npx rhino -f tag -c Tag -d Tag -u Tag -a Tag

Add your imports, and you are good to go.
